# nuoqin windows开发工具箱.
## 项目介绍
    本项目基于C# WPF制作，主要是想在有无网络的情况下能用一些小工具来解决开发过程中的小问题，就在空闲的时候写了小工具。
    ps:如果您有什么好的建议，请在issues中提出。希望本工具可以帮助到各位！
## 目前本项目支持的功能
- 日期工具
- json格式化
- json转java对象
- json转xml
- md5加密·
- AES加解密
- base64编解码、
- URL编解码
- websocket客户端
- yaml转json
- XML转json.
- 二维码生成
- 二维码打印
- 二维码生成无线网
- 内网工具
- windows系统自带工具
- RSA加解密
- es客户端
- dism++集成
- everything集成
